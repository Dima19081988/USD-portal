# USD Portal

Внутренний портал для врачей УЗИ.

## О проекте

USD Portal — это внутренний веб-портал для врачей ультразвуковой диагностики.
На первом этапе проект ориентирован на работу со структурированными протоколами,
конструктором шаблонов, формированием заключений, а также на образовательный и
информационный контур: материалы, статьи, мероприятия, новости и тренды.

## Цель этапа 1

Основная цель этапа 1 — создать безопасный и расширяемый MVP для внутреннего
использования, который объединяет:

- библиотеку протоколов;
- конструктор шаблонов;
- формирование заключений;
- библиотеку материалов;
- статьи и ссылки;
- мероприятия;
- новости и тренды.

## Технологический стек

- PostgreSQL
- Node.js + TypeScript
- React + TypeScript
- Vite
