# Обзор проекта

## Назначение

USD Portal — внутренний портал для врачей ультразвуковой диагностики, который
объединяет рабочие инструменты для формирования протоколов и заключений,
образовательные материалы и информационные разделы.

## Целевая аудитория

Основная аудитория проекта:
- врачи УЗИ;
- владелец системы;
- в будущем — другие роли при расширении платформы.

## Идея проекта

Проект создается сначала для внутреннего коллектива, но архитектура должна
поддерживать масштабирование в будущем. На первом этапе акцент делается на
структурированные протоколы, шаблоны, формирование заключений и безопасную
организацию доступа.

## Основные контуры системы

### Рабочий контур
- библиотека протоколов;
- конструктор шаблонов;
- создание черновиков и готовых заключений.

### Образовательный контур
- библиотека материалов;
- статьи и ссылки;
- мероприятия.

### Информационный контур
- новости УЗИ;
- тренды УЗИ.

## Архитектурный подход

Проект планируется реализовать как веб-приложение на стеке PostgreSQL,
Node.js + TypeScript, React + TypeScript, Vite. Архитектура должна быть
расширяемой по ролям, модулям и требованиям безопасности.
